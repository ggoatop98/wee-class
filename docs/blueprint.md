# **App Name**: WeeClass Aid

## Core Features:

- Main Navigation: Fixed left-side navigation bar for easy access to different sections.
- Home Screen: Dashboard with quick action buttons for adding appointments and students.
- Student Management: Student list management with add, edit, and delete functionalities.
- Appointment Management: Appointment management with add, edit, and delete functionalities, supporting multiple appointment types.
- Calendar View: Calendar view for visualizing appointments.
- Counseling Record Detail: Detailed student counseling record management with past records and input area.
- Firestore integration: Saving students, appointments and counseling logs into the Firestore.

## Style Guidelines:

- Primary color: Soft blue (#A0CFEC) to evoke a sense of calmness and trust.
- Background color: Light gray (#F0F4F7) for a clean and non-distracting interface.
- Accent color: Muted orange (#DFA878) to highlight important actions and elements.
- Headline font: 'Belleza', sans-serif, for headings and titles.
- Body font: 'Alegreya', serif, for body text and detailed information.
- Use simple, clear icons from a consistent set (e.g., Font Awesome) to represent navigation items and actions.
- Responsive layout design that adapts to different screen sizes (desktop, tablet, mobile).
- Subtle transitions and animations to provide feedback on user interactions (e.g., modal opening, data saving).
